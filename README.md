Presidential Innovation Fellows Foundation website
===================

### Requirements

- Jekyll
- GitHub Pages
- Zurb Foundation 5
- SASS (SCSS)

### build it

- fork this repo
- clone your version of it

```
cd presidential-innovation-foundation.github.io
bundle install
```
